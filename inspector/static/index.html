Playback

Space

← / →
Seek ±3 s
↑ / ↓
Jump to prev / next word boundary
, / .
Slower / faster playback

Navigation

[ / ]

R
Random verse (current reciter)
Shift+R
Random verse (any reciter)
J
Scroll active word into view

Display

A
Toggle Analysis / Animation view
L
Toggle letters (analysis) or words mode (animation)
P
Toggle phonemes (analysis) or letters mode (animation)

Click waveform
Seek to that position
